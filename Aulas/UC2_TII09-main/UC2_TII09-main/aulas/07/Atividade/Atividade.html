Síntese das propriedades CSS


  1. color

    2. Define a cor do texto de um elemento. 

     h1 {
    color: blue;
 }

  2. text-align

    3. Alinha o texto dentro de um elemento

    • left: o texto à esquerda (padrão)
    • center: centraliza o texto
    • right: o texto à esquerda (padrão) 
    h1 {
    text-align: Center:
}

  3. font-size

    4. Define o tamanho da fonte de um elemento 

    p { 
    font-size: 18px:
}

  4. background-color

    5. Define a cor do fundo de um elemento 

    button {
    background-color: yellow:
}

  5. color (Para Botões)

    6. Define a cor do texto de um elemento (usado em conjunto com background-color) 

    buton {
    color: black:
}
