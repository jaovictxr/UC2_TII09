HTML

HTML (línguagem de marcação de hipertexto) é o bloco de construção mais básico da web. Define o Significado e a assinatura de conteúdo web.

TAG ou Marcação

O HTML usa "Marcação" para anotar texto, imagem e outros conteúdos para exibição em um navegador web.

Um elemento HTML é separado de um outro texto em um documento, por "tags". As tags consistem no nome do elemento entre um "<" e ">".
